# drcassignment
Computer Inventory Management

### Installation
Follow these command to start projec in local machine.

```sh
$ git clone https://github.com/ketanvnavadiya2/drcassignment.git
$ cd drcassignment
$ npm install
$ ng serve
```

Open the browser and enter below URL

```sh
$ http://localhost:4200/
```
